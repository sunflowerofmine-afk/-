2. 3가지 패턴
시스템이 인식하는 매수 후보 패턴은 총 3종류입니다. 공통 전제 조건: 최근 1~3거래일 내에 장대양봉(기준봉)이 발생해야 합니다.
🚀 당일돌파형
오늘 장대양봉이 발생한 경우.
기준봉 = 오늘이므로 즉시 진입을 검토합니다.
오늘 몸통 비율 ≥ 15% (준장대 10%)
윗꼬리 ≤ 전체 범위의 25%
거래대금 1,500억 이상
교집합(상승률 Top20 ∩ 거래대금 Top20) 여부 확인
🔶 고가수축형
1~3일 전 장대양봉 후 거래대금이 크게 줄고 주가가 기준봉 고가 근처에서 좁게 유지되는 패턴.
"물량 소화 완료 → 재점화 대기" 상태입니다.
기준봉 이후 평균 거래대금 ≤ 기준봉의 50%
오늘 거래대금 ≤ 기준봉의 40%
고가~저가 변동폭 ≤ 8%
오늘 종가 ≥ 기준봉 고가 −5%
📊 고가횡보형
1~3일 전 장대양봉 후 주가가 기준봉 고가 −5% 이내에서 유지되는 패턴. 고가수축형보다 조건이 덜 엄격합니다.
1~3일 전 기준봉 발생
오늘 종가 ≥ 기준봉 고가 −5%
구조 붕괴(−8% 초과) 없음
우선순위: 당일돌파형 > 고가수축형 > 고가횡보형
한 종목이 여러 조건을 동시에 만족하면 우선순위가 높은 패턴으로 분류됩니다.
3. 핵심 개념
기준봉 Base Candle
패턴의 출발점이 되는 장대양봉 캔들입니다. 아래 조건을 동시에 만족해야 기준봉으로 인정됩니다.
• 몸통(시가~종가) 비율이 캔들 전체 범위의 15% 이상
• 당일 거래대금이 이전 20일 평균 대비 3배 이상 폭발
이 "폭발적 거래대금 + 장대양봉" 조합이 핵심입니다.
교집합 Intersection
상승률 Top20과 거래대금 Top20 양쪽에 동시에 이름을 올린 종목입니다. 오르면서 동시에 거래가 많다는 의미 → 시장 참여자가 집중하는 종목.
교집합에 속한 종목에는 ★교집합 태그가 붙습니다.
구조 붕괴 Structure Breakdown
기준봉 발생 후 어느 날 종가가 기준봉 고가 대비 −8% 초과 하락한 경우. 이 종목은 패턴 조건을 상실하며 후보에서 제외됩니다. 눌림이 너무 깊으면 매도 세력이 강하다는 신호로 해석합니다.
거래대금 ratio TV Ratio
오늘 거래대금 ÷ 기준봉 거래대금의 비율입니다.
• 1.0 = 기준봉과 같은 수준
• 0.4 = 기준봉 대비 40% (수축 진행 중)
• 0.05 미만 = 너무 급감 → 관심 소멸 의심으로 제외
고가수축형/횡보형은 ratio가 낮을수록 "물량 소화 완료"에 가깝습니다.
고가대비 Gap from Base High
(오늘 종가 − 기준봉 고가) ÷ 기준봉 고가 × 100
예) +1.2% → 기준봉 고가를 돌파한 상태
예) −4.5% → 기준봉 고가보다 4.5% 아래에서 마감
−5% 이내 유지면 고가횡보형, −8% 초과이면 구조 붕괴 판정.
기준봉 오프셋 Base Candle Offset
기준봉이 몇 거래일 전에 발생했는지를 나타냅니다.
당일(0) → 오늘이 기준봉 (= 당일돌파형)
1일전(1) → 어제 기준봉
2일전(2), 3일전(3) → 고가수축형·횡보형에 해당
recent_base_pool 관찰 풀
과거 5거래일의 신호 데이터에서 당일돌파형이었던 종목을 추출해 오늘 다시 점검하는 후보 풀입니다.
오늘 Top20에 들지 못해도 거래대금 300억 이상이면 관찰 대상에 포함됩니다. 고가수축형·고가횡보형 탐지를 위한 핵심 메커니즘입니다.
4. 시장 지표
용어설명
ADL Advance-Decline Line의 약자. 상승 종목 수 ÷ (상승+하락 종목 수) 비율입니다.
55% 초과 → 강세 판정 / 40% 미만 → 약세 판정
지수가 올라도 ADL이 낮으면 소수 대형주만 끌어올린 것일 수 있습니다.
강세 / 중립 / 약세 시스템이 판단하는 당일 시장 수용성입니다.
🟢 강세: ADL > 55% AND 1,500억 이상 종목 3개↑
⚪ 중립: 강세·약세 조건 미해당
🔴 약세: ADL < 40%
자금집중형 약세 장이지만 지수는 올라 있는 상황.
특정 대형주·주도 섹터에 자금이 쏠리고 나머지는 빠지는 패턴입니다.
이 경우 후보는 더 보수적으로 압축됩니다.
1,500억↑ N개 당일 거래대금이 1,500억 원 이상인 종목 수입니다.
숫자가 많을수록 시장 전반에 자금이 활발히 유입 중임을 의미합니다.
교집합 N개 상승률 Top20 ∩ 거래대금 Top20 동시 진입 종목 수입니다.
이 숫자가 많을수록 오른 종목에 거래가 집중되는 효율적 장세입니다.
상한가 등락률 +29.5% 이상 종목을 상한가로 분류합니다.
이미 급등했으므로 종가베팅 진입 불가 → 후보에서 자동 제외됩니다.
5. 수급 용어
용어설명
기관 순매수 당일 기관투자자(투신·보험·연기금 등) 합산 순매수 금액(원화)입니다.
양수(+)면 매수 우위, 음수(−)면 매도 우위입니다.
외국인 순매수 당일 외국인 투자자 합산 순매수 금액입니다.
5d (5일 누적) 최근 5거래일 합산 순매수 금액입니다.
당일이 매도라도 5일 누적이 매수면 추세적 수급 관점에서 긍정적으로 봅니다.
수급 라벨 기관·외국인 매수 현황을 한 줄로 요약한 태그입니다.
예) 기관+외인매수 / 기관만매수 / 매도우위 등
프로그램 순매수 차익·비차익 프로그램 매매를 합산한 순매수입니다.
2차(17:50) 실행 시 장 마감 후 확정치를 수집합니다.
6. 체크리스트 (N/4)
각 종목에 대해 4가지 필수 조건을 점검합니다. 3개 이상 통과 시 핵심 후보로 분류됩니다. 대시보드에서 체크(3/4) 형태로 표시됩니다.
기호조건명설명
大 (장대양봉) 오늘 장대양봉 여부 오늘 몸통 비율 ≥ 10%, 윗꼬리 ≤ 25%. 당일돌파형의 핵심 조건.
첫봉 (첫 장대양봉) 최근 30일 내 첫 장대양봉 최근 30일 내 동일한 수준의 장대양봉이 없어야 합니다.
반복 양봉은 추가 상승 여력이 소진된 신호일 수 있습니다.
MA (이평밀집) 이동평균선 밀집 5일·10일·20일·60일 이동평균선이 서로 5~8% 이내에 모여 있는 상태.
이평선이 밀집되면 방향성 돌파 시 큰 움직임이 나오는 경향이 있습니다.
대금 (거래대금) 거래대금 1,500억 이상 당일 거래대금이 1,500억 원 이상이어야 합니다.
유동성이 충분해야 진입·청산이 원활합니다.
추가 참고 항목 (체크리스트 외 보조 정보)
Peak: 최근 60일 대비 거래량/거래대금 신고점 여부
